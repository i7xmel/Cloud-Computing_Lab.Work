# Cloud Computing Lab Work

This repository contains practical implementations of core cloud computing concepts across major cloud platforms including AWS and Google Cloud Platform (GCP). The programs cover Infrastructure as a Service (IaaS), load balancing, auto-scaling, identity management, and cloud-based AI services.

## Programs Overview

### Program 1: Infrastructure as a Service (IaaS) Implementation on AWS
- **Explored IaaS fundamentals**: Virtualized computing resources, scalability, and pay-as-you-go models
- **Compared AWS and GCP services**: Comprehensive analysis of compute and storage offerings
- **AWS EC2 Instance Creation**: Set up Linux virtual machines (2348521_EC2_VM1 and 2348521_EC2_VM2)
- **Security Configuration**: Modified security groups for controlled access management
- **Web Server Deployment**: Installed and configured Apache HTTP Server
- **Website Hosting**: Created HTML pages and hosted websites accessible via public IP addresses
- **Basic Programming**: Implemented "Hello World" application on cloud instances
